MACC Roofing

April 2, 2017 http://maccroofing.com

This is a project I did for a client that needed an online presence for his roofing business. I built him a static website using the GOlang Hugo generator, and hosted it on github pages. I than integrated it with Forestry.io to allow for a CMS like expeprience. He can now update content on the fly without having to pay a monthly fee.
